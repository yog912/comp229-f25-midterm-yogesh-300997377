Games API

Student: Yogesh (300997377)

Endpoints

  • GET /api/games
  • GET /api/games/filter?genre=Action
  • GET /api/games/0
  • POST /api/games
  • PUT /api/games/:id
  • DELETE /api/games/:id

Quick Start

npm install
npm start
# visit http://localhost:3000

Sample cURL

curl -i http://localhost:3000/api/games
curl -i "http://localhost:3000/api/games/filter?genre=rpg"
curl -i http://localhost:3000/api/games/0
curl -i -X POST http://localhost:3000/api/games \
  -H "Content-Type: application/json" \
  -d '{"title":"Hades","genre":"Roguelike","platform":"PC","year":2020,"developer":"Supergiant Games"}'
curl -i -X PUT http://localhost:3000/api/games/6 \
  -H "Content-Type: application/json" \
  -d '{"title":"Baldur'\''s Gate 3","genre":"RPG","platform":"PC","year":2023,"developer":"Larian Studios"}'
curl -i -X DELETE http://localhost:3000/api/games/6
